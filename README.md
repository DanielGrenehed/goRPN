# goRPN
 Reverse Polish Notation implementation in Go 

## Running
run   
``` 
go run main.go
```
to enter interactive mode   
   
```
go run main.go return6.rpn
```
to run the operations from the .rpn file return6.rpn   
   
```
go run main.go 1 4 +
```
to add 1 and 4   
   
```
go run main.go "4 2 *"
```
to multiply 4 by 2 without the need for backslashing the star
